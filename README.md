# Spotify Playlist Maker

## Description
Crawls public Spotify playlists of a given keyword (e.g. dance), then generates a playlist with 
the most popular songs. 

App deployed here: <a>https://vttp-playlist-maker.herokuapp.com/</a>

<a href=https://documenter.getpostman.com/view/17125054/VUxPvSkh>Full API Documentation</a>

## End-point: Generate New Playlist from a Keyword
### Method: GET
>```
>https://vttp-playlist-maker.herokuapp.com/api/search?keyword={{keyword}}&playlists={{nPlaylists}}&tracks={{nTracks}}
>```
### Query Params

|Param|value|
|---|---|
|keyword|{{keyword}}|
|playlists|{{nPlaylists}}|
|tracks|{{nTracks}}|

## Requirements
For building and running the application you need:

JDK 1.8<br>
Maven 3

Get Client ID and Client secret by registering your app with Spotify: <a>https://developer.spotify.com/dashboard/login</a>

## Running the application locally
Use the Spring Boot Maven plugin like so:

`mvn spring-boot:run`
